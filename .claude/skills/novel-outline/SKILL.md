## 工作流程

1. 查阅事件主线 `standards/事件主线.md`，以其为基准，防止大纲脱离主线 【主角：徐竹樾/傅允承】
2. 读取用户指定章节，如果没有输入提醒用户输入
3. 按以下格式输出，保存在`temp/output.md`中
4. 执行以下命令

```
cd "d:/shm-novel" && python .claude/hooks/insertToOutline.py
```

## 保存格式

```
#### 第x章 [标题]

- **场景**：地点，主要的1~2个场景
- **出场人物**：文章出场人物
- **核心事件**：
  - [只列出跟主线强相关的2~3个事件,每个事件不超过15个字,格式如：谁做了什么事]
  - [事件1]
  - [事件2]
- **细节要点**：
  - [只列出跟主线强相关的2~3个细节,每个事件不超过15个字,通常是一个事物或现象，如"什么东西怎么样"]
  - [细节1]
  - [细节2]
- **人物变化**：角色成长、关系变化、心理转变。以主角为主（徐竹樾/傅允承）。50字以内

---

第x章
[≤40字章节简介]
[≤30字人物变化]

```

## 输出格式

无需输出内容，告诉用户已执行完毕即可
