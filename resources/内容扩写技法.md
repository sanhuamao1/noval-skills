## 扩充实操流程

### 第一步：诊断不足原因

对照前面的诊断表格，判断当前章节最薄弱的环节在哪里。可能有多个问题并存，优先解决最严重的那个。

| 问题     | 特征                    | 推荐策略     |
| -------- | ----------------------- | ------------ |
| 场景太薄 | 只有对话，没有环境/感官 | 场景肌理充实 |
| 跳跃太快 | 关键场景一笔带过        | 关键时刻放慢 |
| 情感平面 | 读者感受不到角色情绪    | 内心世界展开 |
| 事件稀少 | 本章只有一个事件        | 次要情节穿插 |
| 对话干瘪 | 对话只传递信息          | 对话层次丰富 |

### 第二步：选择匹配的扩充技巧

根据诊断结果选择一至三种技巧。注意不要在同一段落叠加太多手法

| 诊断结果 | 首选技巧     | 可叠加技巧   |
| -------- | ------------ | ------------ |
| 场景太薄 | 场景肌理充实 | 感官维度叠加 |
| 跳跃太快 | 关键时刻放慢 | 内心世界展开 |
| 情感平面 | 内心世界展开 | 对话层次丰富 |
| 事件稀少 | 次要情节穿插 | 场景肌理充实 |
| 对话干瘪 | 对话层次丰富 | 内心世界展开 |

### 第三步：找到最佳插入点

找到叙事节奏中自然的"喘息点"——场景转换处、对话间隙、角色独处的时刻——在这些位置插入扩充内容，让扩充成为节奏的一部分。

## 扩充技巧

### 1. 充实场景或描写

1. **感官层**：视觉+听觉+触觉+嗅觉，至少覆盖两到三种
2. **空间层**：角色在空间中的位置、移动、与其他物体的距离关系
3. **情绪层**：环境如何映射或反衬角色当前的心理状态

#### 感官维度检查清单

| 感官 | 适用场景           | 常用元素                               |
| ---- | ------------------ | -------------------------------------- |
| 视觉 | 所有场景           | 颜色、光影、明暗、形状、动态           |
| 听觉 | 所有场景           | 人声、环境音、音乐、沉默、节奏         |
| 触觉 | 动作/亲密/恐惧     | 温度、质感、疼痛、压力、湿润/干燥      |
| 嗅觉 | 封闭空间/食物/恐惧 | 食物香气、体味、腐败、金属味、烟味     |
| 味觉 | 进食/受伤/亲密     | 食物味道、血腥味、苦涩（情绪化的味觉） |

**原始：**

> 他走进房间。

**扩写后**

> 门锁是完好的，没有撬痕。李铭推开门的时候，一股难闻味道扑面而来。他下意识屏住呼吸，侧身挤进门缝。
> 这间卧室只有十平米出头，一张一米五的床就占了三分之一。床单是浅粉色的。女尸仰面躺在床上，双手交叠放在腹部，头发梳得很整齐。
> 窗外是六楼的落差，底下是水泥地面。如果是跳窗逃跑，凶手要么身手极好，要么根本没打算从窗户走。
> 李铭注意到床头柜上的水杯，里面还有小半杯水，水面平静，没有灰尘，说明是最近才倒的。

**扩充要点：**
嗅觉（难闻）
→ 空间感（十平米、床占三分之一）
→ 物品细节（粉色床单、整齐的头发）
→ 反常细节引起推理（头发太整齐、楼高）
每一处细节同时服务于悬疑氛围和线索铺设。

### 2. 关键时刻放慢

1. **动作分解**：把一个笼统动作拆成"准备—执行—意外—调整—完成"的微步骤序列。
2. **思维展开**：在动作的间隙插入角色的判断、回忆、预判和情感波动。

**原始：**

> 方旭翻到日记的最后一页，看到了那个名字。他整个人僵住了——凶手竟然是每天和他一起吃饭的搭档。

**扩写后**

> 那页日记写的是很日常的东西，菜市场、天气、一档电视节目的感想。他无意识地翻向下一页，
> 然后他看到了那个名字。
> 不是出现在正文中，是写在页脚的一行小字，像是写日记的时候突然想起什么、随手记下的。
> 方旭的手停在半空中，纸页被他捏住的边角开始微微颤抖。他没有立刻想到"凶手"这个词——大脑拒绝这么快建立联系。他先想到的是今天中午，那个人坐在他对面吃饭，筷子夹着红烧肉递到他碗里说多吃点。
> 这些日常画面和页脚那个名字慢慢连接起来。到所有线连上的时候，他不得不坐在椅子上，因为腿已经不听使唤了。

**扩写要点：**
翻页前后的反差（日常vs真相）
→ 细节发现的过程（位置）
→ 大脑的拒绝反应（先想到的是日常画面）
→ 理智连线的逐步过程（联系-腿软）
恐惧不是瞬间到达的，而是一步步逼近的。

### 3. 内心世界展开

内心戏不是简单地写"他很高兴"或"她很害怕"。

**1. 思维链展开**：一个念头引出下一个，形成链条
**2. 感官触发记忆**：感官自然地带出回忆，比直接说"他想起了..."自然得多。
**3. 内心对话**：角色在自己的脑子里和自己吵架——自我质疑、自我安慰、自我欺骗。
**4. 身体反应映射心理**：不直接写情绪，而是写身体

**示例**

> 方旭把日记合上之后，坐在椅子上没动。他在想一件事——如果搭档是凶手，那上个月他们一起审讯嫌疑人的时候，搭档突然岔开话题、不让嫌疑人继续说下去那次，就不是巧合了。
> 不对，他对自己说，不要这么快下结论。也许只是巧合。也许名字相同。全国叫这个名字的人没有一千也有八百。
> 但他知道不是。因为那个名字旁边还写着四个字，"老地方见"。他们两个人的"老地方"——局门口左拐那个早餐铺子，搭档每周至少约他去三次。每一次都是搭档主动约。每一次方旭都以为是关心。
> 现在他不得不重新审视每一次"关心"。上周四搭档突然来他家送文件，是巧合还是来确认什么？上上周搭档在他车里等他，副驾驶的抽屉被翻过吗？他的记事本在抽屉里。记事本上有所有线人的联系方式。
> 方旭站起来走到窗边。他需要冷静。他深吸了一口气，发现自己在数——从怀疑搭档到现在，他已经在同一个位置坐了多久？如果搭档在外面观察他，会不会通过窗户看到他的异常反应？
> 他拉上了窗帘。

### 4. 对话层次丰富

丰富的对话同时在做很多事：传递信息、展现性格、制造张力、暗示潜台词、推进关系。
**1. 动作和表情中断**：人在说话时会做其他事——比如喝水、翻文件、看手机、整理衣领。它们透露说话人的真实状态：紧张、不感兴趣、在思考如何措辞。
**2. 潜台词层**：角色说出口的话和他真正想说的不是一回事。读者需要从上下文、表情、语气推断真实含义。
**3. 话题绕弯再回来**：现实中人们很少直接切入正题。先聊天气、聊食物、聊一个无关的事，然后突然拐回核心话题。
**4. 对话中的权力博弈**：谁先开口、谁沉默、谁转移话题、谁追问，每一个对话行为都是权力关系的微调。

**原始：**

> "好久不见。"陈默说。
> "是啊，三年了吧。"林可笑了笑，"你看起来挺好的。"
> "还行。你呢？工作怎么样？"
> "挺好的。"
> "那就好。"
>
> 两个人沉默了一会儿。陈默终于说："其实我今天找你，是有件事想跟你说。"
> "什么事？"

**扩写后：**

> 三年没见，她瘦了。这是他注意到的第一件事。然后他注意到她换了发型，短发，比以前利落，但眼睛下面有青黑色的影子，粉底没盖住。
> "好久不见。"她在他对面坐下，把包放在旁边的空椅子上。动作很自然，像以前在他们经常去的那个大排档一样——先把包放好，再点单，然后才开始说话。
> "三年了。"陈默说。他本来想说"你瘦了"，但在开口前改了主意。这句话太像寒暄了，而他今天来找她不是为了寒暄。
> "你看起来还好。"林可拿起菜单，低头看了一眼，又放下了。她没看菜单，他看出来了——她的目光在菜单上停留的时间不够读完一行字。她在回避什么，或者她在给他时间组织语言。
> "工作还行？"
> "嗯。还行。"
> "那就好。"
> 服务员过来点单的时候，两个人同时看向菜单，这个短暂的转向像是一种默契的喘息。林可要了拿铁，陈默说"再来一杯美式"，然后服务员走了，借口也没了。
> "你怎么突然约我？"林可先开口了。
> "嗯……"他把杯子又转了半圈，"其实是有一件事。"
> "你说。"
> "你爸……"他停了一下。林可的手臂抱得更紧了，指尖掐进了上臂的布料里。她知道他要说什么了。陈默看到她的反应，忽然觉得自己准备了一路的措辞全部作废——她早就知道了，她只是在等别人亲口说出来。

**对比分析：** 多出了：

- 陈默对林可外貌变化的观察（瘦了、黑眼圈）
  → 双方的回避策略（假看菜单）
  → 肢体语言暗示紧张（转杯子）
  → 林可"先开口"这个行为本身就是权力博弈（谁先问，谁就更迫切）
  → 最后的暗示（她已经知道了）

### 5. 次要情节穿插

次要情节不是打断主线，而是像编织中的暗线，从主线情节的缝隙里穿过去。好的次要情节让章节更丰满，同时为后续章节埋下种子。

**1. 配角的小故事片段**：不用给配角完整的故事线，只需要在主场景中给他一个细节，让读者觉得这个人也有自己的生活在继续。
**2. 暗线的一笔推进**：在主场景中用一两句提到某个之前埋下的伏笔或暗线，保持它在读者记忆中的活跃度。
**3. 伏笔的自然埋设**：以不经意的方式提到某个人、某件事、某个物品，后面再揭示它的重要性。
**4. 关系的微妙变化信号**：角色之间的关系在每一章都应该有微小的变化，哪怕只是一个眼神、一个动作的差异。

**示例：配角片段——悬疑中的搭档日常**

> （主线：方旭正在分析案情，沉浸在自己的思考中）
>
> 桌上手机亮了一下，是搭档发来的消息："今晚吃火锅？我请。"后面跟了一个表情包，一只企鹅举着辣椒，晃来晃去的。方旭盯着那只企鹅看了三秒——搭档以前不发表情包，是从什么时候开始用的？他想不起来。
>
> 他没有回复，把手机翻过去扣在桌上。继续看手里的案件资料。
>
> （注：这条消息和搭档的"日常化"行为，在后面揭晓搭档身份时会让读者回过头来细品：每一次"关心"都是监控，每一个表情包都是降低防备的手段。）

## 扩写检查清单

**防注水原则**
以下扩充是无效的，必须避免：

- **无意义的风景描写**——与角色情绪或情节走向无关的景色堆砌
- **凑字数的回忆**——不推进人物弧光的回忆段落
- **重复已知信息**——换个说法把读者已经知道的事再说一遍
- **过渡冗长**——不必要的路程描写、等待描写、日常流水账
- **对话注水**——大量"嗯""啊""哦"和毫无信息量的寒暄

**扩写禁忌**

1. **不要为了字数而扩写**：每个新增细节都要有目的
2. **不要重复解释**：相信读者的理解力
3. **不要偏离主线**：扩写是为了深化，不是偏离
4. **不要过度描写**：保持简洁是美德

扩写后检查：

- [ ] 是否保持了原文的核心信息？
- [ ] 新增内容是否服务于氛围或情节？
- [ ] 节奏是否合适？（不应拖沓）
- [ ] 是否符合克制·留白原则？
- [ ] 视角是否一致？
