# 写作技法与节奏

## 核心原则

### 白描为主

- 名词和动词为主，形容词能砍就砍
- 一句话最多一个修饰语
- 动作代替形容："他老了"不如"他扶着墙站起来"
- 写完一段后，删掉所有形容词再读一遍，如果意思没变，那些形容词就是多余的
- 优先使用具体、可感的细节
- 避免陈词滥调的比喻， 比喻只在能准确传达独特感受时使用

### 克制·留白

- **不直接表达情感和心理状态**：通过动作、环境、对话间隙呈现
- **信任读者**：不解释情节，让读者自己理解
- **留白**：不说尽，给读者想象空间
  - 情感场景：写动作，不写感受。"她把信折好放回信封"比"她心如刀绞"有力
  - 转折场景：写到转折发生的那一瞬间停止，下一章再接
  - 高潮场景：最高潮处戛然而止，不要写角色的感想和总结

### 短段落为主

- 一个场景一个段落
- 紧张时一句话一段
- 避免大段描写

## 其他描写技法

### 蒙太奇剪辑

用锐利的场景跳跃制造意义。不做平滑过渡，让并列本身讲故事。

**操作要点：**

- 两个场景之间直接切换，不用"与此同时""另一边"等过渡语
- 用空行或分隔符标记跳跃
- 选择有强烈对比或呼应的两组画面
- 对比产生第三层含义：A+B=C（C不在文本里，在读者脑子里）

**示例**

```
 一场婚礼祝酒直接切到一场葬礼。不需要任何解释文字，对比本身就告诉读者：有人从喜到悲，时间流逝，物是人非。
```

## 节奏控制

### 紧张场景

- **短句**：碎片化，加速节奏
- **感官聚焦**：减少细节，突出关键感官
- **即时性**：减少过渡，直接呈现

### 情感场景

- **放慢节奏**：延长动作，增加细节
- **环境反射**：用环境烘托情绪
- **对话间隙**：利用停顿和沉默

### 过渡场景

- **简洁**：快速带过，不拖沓
- **功能性**：服务于情节推进
- **时间跳跃**：明确但不用力解释

## 开头技巧

不一定要遵循，符合章节内容再遵循。

### 有效切入

- **动作切入**：直接呈现正在发生的事
- **对话切入**：用对话建立场景和冲突
- **场景切入**：用独特的环境氛围引入
- **倒计时切入**：（例如：天亮之前，密函必须送到。城门一开，里面的人全都要死。）
- **重大发现切入**
- **危机时刻切入**
- **谜团切入**
- **重大选择切入**：（例如：救生艇只能载一个人。她的丈夫和女儿都在冰冷的海水里挣扎。她伸出手——）

### 避免

- 大段背景介绍
- 人物外貌静态描写
- 天气/时间/地点的流水账

### 首句原则

- 有画面感或悬念感
- 建立基调
- 让读者想问"然后呢"

## 对话规范

### 对话的功能

- 推进情节
- 展现性格
- 暗示关系
- 埋设信息

### 对话的节奏

- 打断和重叠（紧张时）
- 停顿和沉默（情感时）
- 一人说完一人接（日常时）

### 对话标签

- 用动作替代标签
- 避免过多副词修饰
- 优先使用"说"

## 禁止事项

1. **解释情节**：不写"这是因为..."
2. **标签化情感**：不写"他感到悲伤"
3. **过度内心**：不写大段心理活动
4. **上帝视角**：不写角色不知道的信息
5. **陈词滥调**：避免"像樱花一样美丽"之类的表达
