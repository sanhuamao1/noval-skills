## 核心主题

> 用 1-3 句话概括故事的核心主题。可参考以下句式：
>
> - "这是一个关于 **\_** 的故事。"
> - "**\_** 与 **\_** 在 **\_** 中 **\_**，最终 **\_**。"
>
> 示例："两个童年相遇过的人，成年后以不同身份卷入同一场漩涡，最后并肩与成长。"

## 事件主线

> 在此列出故事的核心事件序列（可用 bullet list 或编号）。
> 可选方式：
>
> - **三幕式**：起因 → 发展 → 高潮/结局
> - **节拍表**：按事件时间线线性排列
> - **线索式**：按不同线索分别列出

## 情感弧线

> 记录主角/关键角色的情感变化阶段。表格行可自由增删。

| 阶段 | 状态           | 核心冲突           | 标志事件 | 内心变化           |
| ---- | -------------- | ------------------ | -------- | ------------------ |
|      | 当前关系的状态 | 推动阶段变化的矛盾 | 触发事件 | 主角此时的内心活动 |
|      |                |                    |          |                    |

### 填写提示

- **阶段**：用两个字概括，如"初遇期""怀疑期""卷入期"
- **状态**：此时角色之间的位置关系
- **核心冲突**：推动这一阶段发展的内在矛盾
- **标志事件**：这一阶段最重要的一到两个事件
- **内心变化**：主角在情感上的认知或态度转变
